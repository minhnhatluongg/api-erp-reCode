1. Tạo và gọi URLProtocol để truyền request và mở app ký client:	
	-> Cấu trúc jwt:
		payload_string:  join(';', [
							'xml_all',			//0 - fix value
							<keyUID>,			//1 - key process UID
							<SignerTaxcode>,	//2 - Taxcode to sign
							'%',				//3 - fix value
							'hopdong_b',		//4 - fix value
							<API_base_url>,		//5 - BaseURL,
							<api_tiếp_nhận_trạng_thái>,    //6 - ChangeStatusAction
							'0'             //7 - fix value
						]);
	-> Tạo JWT với secrect_key
	-> Tạo request URL, cấu trúc:  MoSignature:<jwt>
	
2. Các API Cần có:
	a. API tiếp nhận trạng thái xử lý:
		SignApp call api, đẩy trạng thái xử lý ký, post data:
			{
				processName,	// = keyUID,
				status,			// status
                message, 		// message 
                data 			// other data
			}
	b. API Valid JWT:
		Kiểm tra chuỗi jwt hợp lệ
		- action: /api/process/valid_jwt
		- post data: 
			{
				jwt : <jwt> //chuỗi jwt
			}
		- response:
		    {
				isSuccess: true|false,
			    data: {
					apiInfo: <apiInfo>,
				},
				message : <string>,
				returnDate: <yyyy/mm/dd HH:i:s>
			}
		* Lưu ý:
			Giá trị apiInfo: chuỗi bos_encrypt ( <api_user_name>-_-<api_password> )
	c. API lấy data request:
		Lấy dữ liệu yêu cầu xử lý:
		- action: /api/invoice/get_inv_param
		- post data:
			{
				keyID : <keyUID>
			}
		- response:
			{
				isSuccess: true|false,
			    data: [				//array
					invOID: <oid_hợp_đồng_ký>,
				],
				message : <string>,
				returnDate: <yyyy/mm/dd HH:i:s>
			}
	d. API Get XML để ký số:
		Lấy xml để ký số
		- action: /api/invoice/get_xml_all
		- post data:
			{
				oid : <invOID>,
				objType: "hopdong_b" //fix value
			}
		- response:
			{
				isSuccess: true|false,
			    data: {				
					xmlContent: <base_64_encode(xml_hop_dong)>,
					isSigned : 1|0,
					nodeToStore:	<nodeToStore>,
					nodeToSign:		<nodeToSign>,
					nodeSignID:		<nodeSignID>,
					signTime:		<yyyy/mm/dd HH:i:s>		//nếu không cung cấp -> lấy theo ngày giờ trong .returnDate
				},
				message : <string>,
				returnDate: <yyyy/mm/dd HH:i:s>
			}
	e. API Set Signed xml
		Đẩy dữ liệu xml đã ký số lên server
		- action: /api/invoice/get_xml_all
		- post data:
			{
				oid: <oid_hợp_đồng_ký>,
                objType: "hopdong_b" 	//fix value
                xmlContentBase64:	<base_64_encode(xml_hop_dong_da_ky)>,
                appSignVersion: 	<sign_app_client_version>,
                tokenSerial:		<signTokenSerial>
			}
		- response:
			{
				isSuccess: true|false,
			    data: {				
					 maTD : ""  //fix value
				},
				message : <string>,
				returnDate: <yyyy/mm/dd HH:i:s>
			}
